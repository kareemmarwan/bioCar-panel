Build a production-ready Pharmacy Admin Dashboard (Frontend Only) using:

--------------------------------------------------
ARCHITECTURE RULES
--------------------------------------------------

1) No backend logic inside Next.js.
2) Use RTK Query for:
   - Fetching
   - Mutations
   - Caching
3) Use Redux Toolkit for:
   - Auth state
   - User role
   - UI state (sidebar, theme, modals)
4) Use Client Components where needed.
5) Use App Router structure.
6) Implement role-based route protection.
7) Strong TypeScript typing for all models.
8) Separate features by folder.
